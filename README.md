# [Wordminded](https://play.google.com/store/apps/details?id=com.cretihoy.wordminded&pli=1)

Card game for family gatherings and fun parties

## Tech Stack:

- Kotlin
- MVVM
- Jetpack Compose
- Hilt
- State flow
- Coroutines
